## Detailed description

```
QR Code Generator is a lightweight browser extension that turns web content into scannable QR codes in one click.

KEY FEATURES

• Context menu — Right-click on a page, selected text, a link, or an image to generate a QR code instantly
• Toolbar button — Click the extension icon to create a QR code for the current page URL
• Modal preview — View the QR code in a clean overlay with copy and download actions
• Download PNG — Save the QR code as an image file
• Copy content — Copy the encoded text to your clipboard with one click
• Local only — All QR codes are generated on your device; nothing is uploaded
• UTF-8 support — Works with Chinese and other international characters

HOW TO USE

1. Install the extension and visit any webpage
2. Right-click and choose a QR code option, or click the toolbar icon
3. Preview the QR code, then copy the content or download the image

PRIVACY

This extension does not collect, store, or transmit any user data. Processing happens entirely in your browser.

Compatible with Chrome, Edge, Brave, and other Chromium-based browsers.
```
