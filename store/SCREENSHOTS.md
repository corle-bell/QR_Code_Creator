# 商店截图拍摄指南

Chrome 网上应用店要求至少 **1 张截图**，建议准备 **3–5 张** 以提高转化率。

---

## 尺寸要求

| 类型 | 尺寸 | 必填 |
|------|------|------|
| 截图 | **1280 × 800** 或 **640 × 400** | 是（至少 1 张） |
| 小型宣传图块 | 440 × 280 | 否 |
| 顶部宣传图块 | 1400 × 560 | 否 |

推荐使用 **1280 × 800**，在高分屏上更清晰。

---

## 建议拍摄的 5 张截图

### 截图 1：右键菜单（封面图）

**目的**：展示扩展的核心入口

1. 打开任意网页（如 `https://example.com`）
2. 选中一段文字
3. 右键，确保菜单中出现「生成选中文字二维码」
4. 截图包含：网页内容 + 右键菜单

### 截图 2：二维码模态框

**目的**：展示生成结果与 UI

1. 右键选择「生成选中文字二维码」
2. 等待模态框完全显示（含二维码、复制、下载按钮）
3. 截图以模态框为中心，背景网页可略微虚化

### 截图 3：当前页面 URL

**目的**：展示页面级功能

1. 打开一篇新闻或博客文章
2. 在页面空白处右键 →「生成当前页面二维码」
3. 截图模态框，标题显示「二维码 - 当前页面」

### 截图 4：链接二维码

**目的**：展示链接场景

1. 在网页超链接上右键
2. 选择「生成此链接二维码」
3. 截图模态框，内容区显示 URL

### 截图 5：工具栏图标

**目的**：展示快捷操作

1. 点击浏览器工具栏上的扩展图标
2. 截图包含：工具栏图标 + 弹出的二维码模态框

## Windows 拍摄方法

### 方法 A：浏览器窗口调整（推荐）

1. 按 `F12` 打开开发者工具
2. 按 `Ctrl + Shift + M` 进入设备模拟模式
3. 顶部设置自定义尺寸：**1280 × 800**
4. 使用 **Win + Shift + S** 区域截图，或使用 Chrome 全页截图扩展

### 方法 B：固定窗口后截图

1. 将 Chrome 窗口拖到合适大小
2. 使用 Snipping Tool 或 ShareX 截取 **1280 × 800** 区域
3. 在 Paint / Photoshop 中裁剪到精确尺寸

### 方法 C：后期裁剪

1. 先截大图（如 1920×1080）
2. 用任意图片工具裁剪为 1280×800
3. 保存为 PNG（商店接受 PNG 或 JPEG）

---

## 截图质量建议

- 使用**浅色背景**网页，二维码对比更清晰
- 模态框完全加载后再截，避免「生成中...」状态
- 不要包含个人敏感信息（真实邮箱、密码、私人聊天）
- 保持 UI 语言一致（中文扩展用中文界面截图）
- 如需英文商店，可额外准备英文环境截图（本扩展 UI 为中文）

---

## 文件命名与存放

将成品截图放入 `store/screenshots/`：

```
store/screenshots/
├── 01-context-menu.png      (1280×800)
├── 02-modal-preview.png
├── 03-page-url.png
├── 04-link-qr.png
└── 05-toolbar-icon.png
```

上传时在 Developer Dashboard → **Store listing** → **Screenshots** 按功能顺序排列，**第 1 张为封面**。
